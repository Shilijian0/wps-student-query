# wps-student-query
# WPS 表单学生成绩自动查询工具 (WPS-Student-Score-Query-Automation)

基于 Python 的自动化数据查询与爬取工具。本项目利用 `Playwright` 浏览器自动化库，实现了从本地 Excel 读取学生名单，并自动批量登录 WPS 表单查询系统提取学生“排名”、“总学分”和“加权平均分”的功能，最终将结果汇聚导出为标准 CSV 报表。

## 🚀 功能特点

- **批量自动化：** 告别手动逐条输入的低效操作，支持百级别、千级别数据的全自动循环查询。
- **动态防爬拟人化：** 采用 Edge 驱动，内置 `slow_mo` 延迟参数，模拟真实人类输入与点击行为，降低被前端风控拦截的风险。
- **异常容错机制：** 针对网络抖动、信息匹配错误等异常情况加入 `try-catch` 保护，单个学生查询失败不会中断整个队列，确保程序稳定运行。
- **数据清洗防变形：** 自动处理 Excel 导入时的数字精度问题（如手机号、学号长数字变形），确保表单填充 100% 精准。

## 🛠️ 技术栈

- **数据处理：** `Pandas` (用于高效解析本地数据源及生成结构化结果报表)
- **自动化控制：** `Playwright Python` (新一代高性能、支持多内核的现代化全自动测试/爬虫框架)
- **核心内核：** Chromium / Microsoft Edge

## 📁 目录结构建议

为了保障数据安全，运行前请确保本地目录结构如下：
├── data/

│   └── students.xlsx       # 本地学生数据源（包含“学生学号”、“姓名”、“手机号”列）

├── output/

│   └── 查询结果.csv         # 运行后自动生成的查询结果报表

└── query_script.py         # 本核心执行脚本
